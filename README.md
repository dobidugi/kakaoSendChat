# 카카오톡 채팅 전송 프로그램

##### =============================================================================
##### 해당 프로그램은 프로그램에 메세지를 입력시 열려있는 모든채팅방에 전송해줍니다.
##### 해당 프로그램은 프로그램 실행도중에 원하는 창을 닫아도  
##### 메세지를 닫은창에 메세지를 보내지를 전송하지 않습니다.
##### 해당 프로그램은 프로그램 실행도중에 원하는 추가로 열어도  
##### 해당창에도 메세지를 보낼수있습니다.
